Tensorflow windows 11 설치 가이드

사전 설치​

WSL2 설치​

tensorflow를 설치하는 곳은 단순한 windows가 아닌 windows 내부에 만들어질 가상머신입니다. 따라서, wsl2를 먼저 설정해 주어야 합니다.

파이썬​

파이썬 설치​

info

tensorflow 호환 가능한 파이썬 3.9-3.11 버전 설치

링크를 통해 파이썬 3.9.13 버전 windows installer 설치

파이썬 설치 확인​

python --version
Python 3.9.13

pip 설치 여부 확인​

note

pip는 운영체제 파이썬 기본 설치 패키지. 만약, python 3.9.13만 설치되어 있다면 pip 및 pip3 명령 모두 동일한 python 3.9.13에 대한 패키지를 관리

pip --version

NVIDIA 드라이버​

꽤 오래 걸립니다.

NVIDIA 드라이버 설치​

https://developer.nvidia.com/cuda-toolkit-archive

NVIDIA cuDNN 설치​

https://developer.nvidia.com/cudnn

tensorflow 설치​

pip로 설치​

python setup.py egg_info did not run successfully. 에러 대응

pip 업데이트

python -m pip install --upgrade pip

setuptools 업데이트

pip install --upgrade setuptools
python -m pip install tensorflow[and-cuda]

설치 여부 확인​

CPU 설정​

python -c "import tensorflow as tf; print(tf.reduce_sum(tf.random.normal([1000, 1000])))"

아래와 같이 텐서 객체를 반환, 출력하면 정상

tf.Tensor(-2911.9321, shape=(), dtype=float32)

info

cuDNN

python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))"
