## 프로젝트 소개

숭파이(Soongfi)는 숭실대학교 교내 인터넷 문제 해결 앱입니다.

숭실대학교 교내 무선 인터넷은 연결 시 자동 팝업되어야 할 학부생 인증 포탈이 팝업되지 않아 인터넷 인증이 불가능한 문제가 있습니다. 이 문제를 해결하기 위해 API 테스트 툴 Postman을 활용하여 숭실대학교 학부생 인증 포탈을 우회적으로 접속하는 방법을 찾았습니다.

해결 방법을 자동화하여 이를 원클릭 앱으로 개발, 배포하였습니다.
