- 분류 : 모바일 앱

- 스택 : **React-Native**, **Styled-Component** / **Firestore**, **Firebase Authentication**
- 주제 : 독서 기록
- 역할 : 개인 프로젝트

## 프로젝트 소개

체크드(Checked)는 독서 기록 모바일 어플리케이션입니다.

단순한 독서 진도 기록 뿐만 아니라 '금주의 명언', 다른 사용자가 읽은 책을 확인할 수 있는 '사람들이 방금 읽었어요', '큐레이터 추천 도서' 기능을 추가적으로 제공합니다.
