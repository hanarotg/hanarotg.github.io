# Docusaurus zoom image 플러그인 적용

## 개요

이미지를 클릭하면 확대되는 플러그인 설치 방법

## 적용

### 설치

```bash
yarn add docusaurus-plugin-zooming
```

### 적용

`docusaurus.config.js` 파일 수정

```jsx
plugins: [
  //highlight-start
    "docusaurus-plugin-zooming",
  //highlight-end
  ],
```

## 참고 사항

페이지 마지막 이미지만 확대되지 않는 버그가 있습니다.
