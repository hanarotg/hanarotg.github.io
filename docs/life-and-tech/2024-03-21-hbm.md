# HBM (High Bandwidth Memory)

D램을 겹으로 쌓아 공정을 통해 데이터 처리 속도를 높이는 기술

## 일반적인 D램과 HBM 차이

HBM은 미세한 구멍을 통해 크래커 간 데이터 연결 통로를 가진 D램 크래커의 **집합**
