# VLAN

## 개요

:::warning

해당 글은 보강이 필요한 글입니다.

:::

단일 레이어 2계층을 분할한, 논리적으로 구분되는 가상의 LAN

서로 다른 VLAN ID 통신은 3계층 이상에서 구분되어 통신이 가능하다.

## Tagging

:::info

Cisco 장비에서는 trunk로 표현합니다.

:::

다수의 VLAN을 하나의 포트에 사용하는 기술

스위치 ↔ 스위치 간 trunk 구간 통신 시 VLAN ID를 삭제하지 않는 방식으로 전달
