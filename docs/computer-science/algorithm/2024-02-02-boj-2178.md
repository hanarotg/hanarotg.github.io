## 풀이

### 접근

(1,1) 좌표에서 (N,M) 좌표까지 이동하기 위해 필요한 최소 칸 수를 구해야 합니다.

BFS 또는 DFS 기법을 사용해서 최소 칸 수를 구해야 합니다.

(1,1) 좌표에서 탐색을 시작하여 (N,M) 좌표로 이동하면서 최소 칸 수를 갱신해야 하는 과정을 통해서요.

어떻게 탐색해야 할까요? 탐색을 계속 해야 하는 상황을 만족하기 위해서는 탐색할 좌표가 아래의 조건들을 만족해야야 할 겁니다.

- 문제 요구사항에 만족하는 범위애 위치
- 값이 0이 아니며
- 방문하지 않는 좌표

그러나 우리는 위 조건을 만족하는 좌표를 탐색하는 것이 아닙니다. 이 문제의 요구사항은 탐색을 통해 (N, M)좌표까지 이동할 때, 최소 칸 갯수입니다.

- 탐색할 좌표에 저장된 값은 기존 좌표에 저장된 값 + 1 보다 작을 것

만약 탐색할 좌표에 저장된 값이 기존 좌표에 저장된 값 + 1 보다 크거나 같다면 해당 좌표로 가서 탐색할 필요가 없습니다. 최소 이동 칸 수를 구하는 것이 목적이니까요.

이 문제는 BFS로 풀이해야 합니다. [DFS로 푸는 것은 올바르지 않습니다.](https://www.acmicpc.net/board/view/25832)

### 솔루션

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <queue>

using namespace std;
int N, M;
int dx[4] = {0, 1, 0, -1};
int dy[4] = {-1, 0, 1, 0};
vector <vector <int> > MAP(101, vector<int>(101, 0));
vector <vector <bool> > VISITED(101, vector<bool>(101, false));

void bfs() {
    queue <pair<int, int> > Q;
    Q.push(make_pair(0 ,0));
    VISITED[0][0] = true;

    while(!Q.empty()) {
        int x = Q.front().first;
        int y = Q.front().second;

        Q.pop();

        for(int i=0; i<4; i++) {
            int nx = x + dx[i];
            int ny = y + dy[i];

            if(nx < 0 || nx >= N || ny < 0 || ny >= M) {
                continue;
            }

            if(MAP[nx][ny] == 0 || VISITED[nx][ny]) {
                continue;
            }

            if(MAP[x][y] + 1 > MAP[nx][ny]) {
                VISITED[nx][ny] = true;
                MAP[nx][ny] = MAP[x][y] + 1;
                Q.push(make_pair(nx, ny));
            }
        }
    }
}

int main() {
    cin >> N >> M;
    for(int n=0; n<N; n++) {
        string str; cin >> str;
        for(int m=0; m<M; m++) {
            MAP[n][m] = str[m] - '0';
        }
    }

    bfs();

    cout << MAP[N-1][M-1];

    return 0;
}
```
