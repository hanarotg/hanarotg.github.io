**숭파이**는 숭실대학교 교내 WIFI 인터넷 문제를 해결하는 모바일 앱입니다.

- iOS : [https://testflight.apple.com/join/q2qA4inP](https://testflight.apple.com/join/q2qA4inP)
- Android : [https://play.google.com/store/apps/details?id=com.soongfi.soongfi_android](https://play.google.com/store/apps/details?id=com.soongfi.soongfi_android)
- 깃허브 : [https://github.com/soongfi](https://github.com/soongfi)

![noname](https://user-images.githubusercontent.com/34812887/217592422-e9dd6c6a-5654-4b03-be64-71a065dd78a6.png)

> **부연 설명**
>
> 1. 학부생은 교내 WIFI를 사용하기 위해 로그인 포털에서 로그인을 해야 합니다.
> 2. 로그인 포털은 일반적인 방법으로 접속할 수 없습니다.
> 3. 일반적으로 교내 WIFI 접속 시 자동으로 로그인 포털이 호출됩니다.
> 4. 그러나 로그인 포털이 자동으로 호출되지 않는 문제가 빈번히 발생합니다.
> 5. 이때, 로그인이 불가능하여 인터넷을 사용할 수 없습니다.

**숭파이**는 우회적인 방법으로 유효한 로그인 포탈을 강제로 호출합니다.

호출된 로그인 포털을 통해 로그인이 가능하도록 돕습니다.

## 개요

- 개발자 2명
- 기간 : 2022년 11월 - 현재

## 역할

- 교내 로그인 포탈 API 테스트
- IOS 및 android 앱 개발
- 깃허브 그룹 관리

## 사용 기술

- IOS : `swift` `swiftUI`
- android : `kotlin` `jetpack compose`
- tool : `postman`

# 프로젝트 기여 및 배운 점

### API 테스트를 통한 문제 해결

- API 테스트 도구 `postman` 활용
- 교내 로그인 포탈([auth.soongsil.ac.kr](http://auth.soongsil.ac.kr))에 특정 파라미터와 함께 응답 요청
- 다양한 시도를 통해 임의의 유효한 로그인 포탈을 호출할 수 있음을 파악

### IOS 및 android 앱 개발

- `swiftUI` 또는 `jetpack compose` 를 통해 기본적인 화면을 설계하는 방법 이해

### 베타 테스트

- appstore connect 에서 공개 베타 테스트 진행
- 베타 테스터 관리 및 베타 테스트 빌드 관리

### iOS 및 Android 이용자 120명 모집

교내 커뮤니티 홍보 및 학부 동기들 사이의 홍보를 통해 베타 테스터 120명 달성
