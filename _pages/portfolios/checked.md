## 역할

## 사용 기술

- `React Native` `styled-component` `cloud firestore` `firebase authentication` `async-storage`

# 프로젝트 기여 및 배운 점

### 기획부터 개발, 출시 경험

- 제품 프로세스 전 과정 경험

### React Native 앱 개발

- [카카오 도서 API](https://developers.kakao.com/docs/latest/ko/daum-search/dev-guide#search-book)를 활용하여 도서 검색 기능 구현
- `firebase authentication`을 활용하여 구글 및 애플 로그인 기능 구현
- `async-storage`를 사용하여 회원 로그인 여부 확인
- 효율적인 컴포넌트 재사용성 고민 - Atomic Design 적용

### React Native 앱 배포

- IOS 및 안드로이드 앱 심사 과정 경험
